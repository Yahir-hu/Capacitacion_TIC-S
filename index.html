INTRODUCCIÓN

En el encabezado te podrás desplazar a cada uno de los módulos para encontrar que aprenderás en cada uno de ellos con una tabla de ayuda.

A continuación te daré un pequeño resumen de cada uno de los siguientes módulos:


  • Módulo 1: En este módulo trabajamos con MIT App Inventor, el cual es una aplicación online para crear aplicaciones en dispositivos móviles, la cual es muy fácil de usar mediante programación con piezas de rompecabezas, muy intuitivo. También trabajamos con Word y Excel para manejar sus herramientas básicas y visuales, como fómulas, encabezados y pie de páginas

  • Módulo 2: En este módulo creamos redes sociales para una empresa ficticia y convivimos en medios sociales en Internet, y también aprendimos a darle mantenimiento a una computadora, especificamente a su hardware y perífericos.

  • Módulo 3: Empezamos creando diagramas de flujo para después adentrarnos en la programación, en este caso, el idioma fue C++ y llevamos un curso al mismo tiempo de CMU Academy, esto en el submódulo de Programación. En el de Sistemas de Información, usamos varias aplicaciones para crear un sistema de información, el cual es como una aplicación que esta hecha para guardar información y utilizarla cada que lo requerramos.

  • Módulo 4: En este módulo creamos páginas web con editores de código HTML, y también usamos aplicaciones de edición de imágenes como Photoshop y Adobe Illustrator, los cuales son de gran ayuda si somos capaces de entender como utilizarlos.
